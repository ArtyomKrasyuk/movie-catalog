О сайте

Данный сайт является учебным проектом, разработанным для закрепления навыков веб-разработки. Его основная цель — продемонстрировать применение стандартных веб-технологий (HTML, CSS, JavaScript) для создания интерактивного каталога фильмов и сериалов.

Функциональность проекта

  • Поиск и каталогизация: реализована возможность поиска фильмов и сериалов по названию.
  • Фильтрация: предусмотрены фильтры по стране производства, году выпуска и жанру.
  • Сортировка: доступна сортировка по рейтингу и по новизне.
  • Дополнительные разделы: просмотр информации об актёрах, новостной блок с публикациями о киноиндустрии.

Техническая часть

  • Структура сайта построена с использованием HTML5.
  • Для оформления интерфейса применены современные возможности CSS3.
  • Логика фильтрации, сортировки и поиска реализована на JavaScript без использования сторонних фреймворков.

Таким образом, сайт представляет собой учебный пример реализации каталога фильмов и сериалов с базовым набором функций и акцентом на практическое применение ключевых технологий веб-разработки.
